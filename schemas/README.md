# schemas

This folder contains [JSON Schema](https://json-schema.org/) `.json` files for all the shared types in the Aerie project.

| Conversion                                          | Library                                            |
| --------------------------------------------------- | -------------------------------------------------- |
| JSON Schema -> [Java](https://www.oracle.com/java/) | [jsonschema2pojo](http://www.jsonschema2pojo.org/) |  |
