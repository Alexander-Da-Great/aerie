==========
Contribute
==========

We are always looking for those who want to help and appreciate any contribution we can get.

There are two ways to contribute to Aerie :

* `Contribute to the Source Code <https://github.com/NASA-AMMOS/aerie/blob/develop/CONTRIBUTING.md>`_ - over on our GitHub.

  * For instructions on setting up a development environment, see :doc:`our Quickstart guide <quickstart/index>`.
  * For guidelines on contributing to the codebase, see :doc:`here <guidelines/index>`.

* :doc:`Contribute to the documentation <documentation/index>` - by writing new or changing existing Documentation.
